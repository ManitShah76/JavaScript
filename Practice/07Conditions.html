Conditional In JS

The JavaScript if-else statement is used to execute the code whether condition is true or false. There are three forms of if statement in JavaScript

  • If Statement

    • It evaluates the content only if condition is true. The signature of JavaScript if statement is given below.

    Syntax

            if(condition){  
            //content to be evaluated  
            }
  • If else statement

    • The if...else statement executes a statement if a specified condition is truthy. If the condition is falsy, another statement in the optional else clause will be executed.

    Syntax

        if(condition){  
        //content to be evaluated if condition is true  
        }  
        else{  
        //content to be evaluated if condition is false  
        }
  • if else if statement

    • It evaluates the content only if condition is true from several conditions. The signature of JavaScript if else if statement is given below.

    Syntax

            if(condition1){  
            //content to be evaluated if condition1 is true  
            }  
            else if(condition2){  
            //content to be evaluated if condition2 is true  
            }  
            else if(condition3){  
            //content to be evaluated if condition3 is true  
            }  
            else{  
            //content to be evaluated if no condition is true  
            }
