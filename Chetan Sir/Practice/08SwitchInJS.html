Switch In JS

A switch statement can replace multiple if check

It gives a more descriptive way to compare a value with multiple variants.

Syntax

        switch(x) {
            case 'value1':  // if (x === 'value1')
              ...
              [break]
          
            case 'value2':  // if (x === 'value2')
              ...
              [break]
          
            default:
              ...
              [break]
          }
